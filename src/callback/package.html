This package contains a simple XML-RPC client-server callback demo with instructions. It uses the XML-RPC library provided by Apache.

Instructions for running the simple XML-RPC client-server callback demo

  1. Check out this project from CVS.
  2. Compile the project manually or by opening it in Eclipse.
  3. Assume you have chosen two different workstations, ws1 and ws2, preferably next to each other in a lab. (For testing purposes, ws1 and ws2 can be the same.) The server will run on ws1, and the client on ws2.
  4. Open a DOS window on each workstation. To find out the IP (Internet Protocol) Address of a workstation, type 
    ipconfig
    in a DOS window. Let ip1 be the IP Addresses of ws1.
  5. Change to top-level directory of this project on both hosts. On ws1, run 
    java -classpath bin:lib/xmlrpc-1.2-b1.jar callback.ServerImpl
  6. On ws2 (and possibly other workstations), run 
    java -classpath bin:lib/xmlrpc-1.2-b1.jar callback.ClientImpl ip1
    Now you should see some interaction between the client(s) and the server.
